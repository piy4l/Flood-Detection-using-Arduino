To run the simulation:
1) Install Proteus 8.4 SP0 (installation details given in the folder)
2) Copy the library files(arduino, gsm and water sensor) to "C:\Program Files (x86)\Labcenter Electronics\Proteus 8 Professional\LIBRARY\" 
   or, in the LIBRARY folder of the installation directory. 
3) Run the project file and add the hex files in corresponding components(arduino, gsm and water sensor).
4) Run the simulation.
